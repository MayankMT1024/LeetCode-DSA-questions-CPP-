# 🚀 LeetCode & DSA Problem Log

This repository contains my personal collection of LeetCode solutions. It serves as a continuous, ever-growing log of my coding journey as I tackle a wide variety of problems — ranging from fundamental data structures to complex algorithmic challenges.

## 🛠️ Tech Stack & Tools
* **Primary Language:** C++ (Standard Template Library)
* **Environment:** LeetCode Built-in Editor
* **Automation:** LeetHub v2 (Auto-syncs accepted solutions directly from LeetCode)

## 📂 Repository Structure
Solutions are automatically organized into folders by problem name. Each folder typically contains:
* The problem description and difficulty level.
* My accepted C++ solution (`.cpp`).
* Time and Space complexity notes generated at the time of submission.

## 🧠 Topics & Patterns Explored
As I solve random and targeted problems, this repository acts as a database for various DSA concepts, including:
- Arrays & Hashing
- Two Pointers & Sliding Window
- Linked Lists
- Binary Search
- Trees & Graphs
- Dynamic Programming
- Greedy Algorithms

## 📈 Goals
1. **Consistency:** Build a massive, comprehensive database of solved problems over time.
2. **Optimization:** Continuously improve my ability to write clean, edge-case-resistant C++ code.
3. **Efficiency:** Strive for optimal solutions, aiming for the best possible Time (e.g., $O(N)$) and Space (e.g., $O(1)$) complexities.

*This is a living repository, updated automatically as I continue to practice and submit new solutions.*
